# How to test code: 
Make sure to run Ganache on port: 7545 
Clone this repostiory:
```
git clone https://github.com/Alamink/Blockchain-Capstone-master.git
```
Install Dependency:
```
npm install
```
Change directory to eth-contracts :
```
cd .\eth-contracts
```
Compile code :
```
truffle compile
```
Test code : 
```
truffle test
```
